ssh tunnel

先備知識：知道ssh是什麼 使用情境： 遠端有一台沒有提供外網的服務A(資料庫)，且設有Bastion server，如何在本機連上服務A。 解法： ssh -L 1235:target_a_host:4567 bastion_server_host 這樣一來，就可以從localhost:1235連到遠端服務。 可以使用-fN在背景執行 ssh -fN -L [bind_address:]<local_port>:<target_host>:<target_port> remotehost Reference ssh tunnel
